Fairytale Character Dating Website
Aug 2024 - Aug 2024

This project was created for Atlas School. The goal was to put the full-stack skills we recently learned to the test. We were given a theme of "Folklore and Fairytales." With this, my partners and I decided to create a website that mimicked a dating website. We added ten different fairytale characters, with which users could either "heart" (like) or "X" (dislike). In the backend JavaScript, the characters were randomized so that each user sees them in a different order. Once a person "matches" with a character, they are then prompted with the option to chat with the character. Here's where my portion of the project came into play. I implemented an AI chat box, using Claude AI through Amazon Bedrock. Using JavaScript, I provided the interface with descriptions of each character so that it knew the correct way to respond, including but not limited to the length of words to respond with.
